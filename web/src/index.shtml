nMars - Corewars MARS for .NET

nMars is engine, compiler and IDE for programming game called Corewars.

Overview

  • nMars has console interface and graphical IDE
  • Under cover are running Parser, Engine and Debugger components
  • It can be ensembled with Python shell, Unit tests and pMars wrapper components
  • All components share common nMars.RedCode library.
  • You can download several packages.
  • Source code could be checked out from subversion repository or browsed

Features

  • It's well architected, around solid interfaces, extensible and reusable
  • Engine is 100% compatible with pMars and ICWS94
  • Parser is LALR grammar driven
  • nMars is unit tested against pMars
  • Supports most pMars commandline switches
  • IDE supports projects and solutions as a setup frame for your warriors
  • IDE projects containning setup of rules and options
  • IDE has familiar look and feel
  • Graphical debugger is asynchronous to UI, so it can run at any speed, still painting core state.
  • Debugger shell could be extended with IronPython
  • Debugger core views are packed with information about events and status of each cell
  • All configuration is XML driven
  • Target platform is .NET 2.0 or Mono
  • All components but IDE are running on Mono, porting IDE is goal.
  • Licensed under LGPL
